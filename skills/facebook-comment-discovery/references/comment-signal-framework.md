# Comment Signal Framework

Use this when translating an offer into comment-first discovery.

On Facebook, the post or reel is often the surface and the comments are where
audience questions, tool asks, and first-person need show up.

## Strong comment types

- first-person pain
- direct question
- recommendation ask
- tool confusion
- workflow failure
- implementation curiosity

## Weak comment types

- praise
- emoji reaction
- fandom chatter
- generic inspiration response
- inside-joke replies

## What to look for

### 1. First-person pain

Examples:

- `I have the same problem`
- `we still do this manually`
- `this is exactly where I am stuck`

### 2. Direct question

Examples:

- `what do you use`
- `how are you doing this`
- `any recommendations`

### 3. Recommendation ask

Examples:

- `which app`
- `what tool are people using`
- `is there something better for this`

### 4. Workflow failure

Examples:

- `this keeps breaking`
- `ours takes forever`
- `support is terrible`

## Seed strategy

Start from profiles whose audiences are likely to expose the need:

- niche educators
- operators sharing the workflow
- adjacent vendors
- category creators with practical comment sections

## Scoring shortcuts

Up-rank:

- first-person language
- question marks
- named tools
- concrete workflow nouns
- explicit recommendation asks

Down-rank:

- generic praise
- social-only chatter
- fandom behavior
- copied compliments
